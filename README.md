## 🟢 이게 뭔가요?

- **n8n(엔에이트엔)** 은 여러 인터넷 서비스(메일, 메신저, 구글 시트 등)를 자동으로 연결해서, **반복되는 일을 대신 해주는 도구**입니다.
  - 예) "메일이 오면 휴대폰으로 알림 보내기", "매일 아침 뉴스 모아두기".
- **이 키트(검은 창 도구)** 는 그 n8n을 **쉽게 설치·실행·관리**하도록 도와주는 한국어 "리모컨"입니다.
  - 검은 창에 글자가 많이 지나가도 고장이 아니니 안심하세요.

> 한 줄 요약: **`n8n_Manager_One-Click_Install-Run.bat` 파일을 두 번 누르고, 화면 번호를 한 번 누르면 됩니다.**

## ⚡ 빠른 시작 (3단계)

1. **다운로드한 폴더를 엽니다.** 그 안의 `n8n_Manager_One-Click_Install-Run.bat` 파일을 찾습니다.
2. **그 파일을 마우스로 두 번 누릅니다.** 검은 창에 한국어 메뉴가 나옵니다.
   - 만약 **파란색 경고창**(`Windows의 PC 보호`)이 뜨면, 당황하지 말고 👉 [7번 항목](#-처음-실행-시-파란-경고창이-뜨면-꼭-읽기)을 보세요. (한 번만 처리하면 됩니다.)
3. **키보드 `0` 키를 한 번 누릅니다.** (= "처음이세요?") 설치부터 실행까지 자동으로 진행됩니다.

> 잠시 뒤 인터넷 창(브라우저)이 저절로 열리고 **"계정 만들기"** 화면이 나옵니다. 이메일과 비밀번호를 직접 정해 만들면 끝입니다. (이 계정은 **내 컴퓨터 안에서만** 쓰입니다.)

처음 설치는 인터넷 속도에 따라 **1~3분** 걸릴 수 있습니다. 글자가 멈춘 듯 보여도 기다려 주세요.

## ⬇️ 다운로드 방법

이 키트는 **파일 하나(.bat)** 와 **설명서 몇 개**로 이루어져 있습니다. 받는 방법은 어디서 공유받았는지에 따라 다릅니다.

- **압축파일(zip)로 받았다면:** 마우스 오른쪽 클릭 → "압축 풀기" → 풀린 폴더를 엽니다.
- **GitHub 같은 사이트에서 받는다면:** 초록색 `Code` 버튼 → `Download ZIP` → 위와 같이 압축을 풉니다.
- **폴더째 받았다면:** 그대로 사용하면 됩니다.

> 📌 **중요:** `n8n_Manager_One-Click_Install-Run.bat` 와 `사용설명서.txt` 는 **같은 폴더**에 함께 두세요. (메뉴 `[9]`가 같은 폴더의 설명서를 엽니다.)

### 인터넷에서 받은 파일이 "차단"되어 있을 때 (한 번만)

인터넷·메신저로 받은 파일은 Windows가 안전을 위해 **잠가 두는** 경우가 있습니다. 다음처럼 풀어주면 됩니다.

1. `n8n_Manager_One-Click_Install-Run.bat` 파일에서 **마우스 오른쪽 클릭 → "속성"**.
2. 맨 아래에 **"차단 해제(Unblock)"** 라는 작은 네모(체크박스)가 보이면 **체크**합니다.
3. **"확인"** 을 누릅니다. (이 항목이 안 보이면 이미 풀려 있는 것이니 그냥 넘어가세요.)

## 🚀 설치 방법

설치 방법은 **두 가지**입니다. 처음이면 1번을 추천합니다.

### 방법 1) 자동 (추천) — `[0]` 한 번
1. `n8n_Manager_One-Click_Install-Run.bat` 두 번 누르기
2. 키보드 `0` 누르기
3. (Node.js가 없으면) 자동 설치 안내가 나옵니다. 화면 안내대로 따라가면 됩니다.
4. 설치가 끝나면 **자동으로 실행**까지 됩니다.

### 방법 2) 직접 — `[1]` 설치만
1. `.bat` 두 번 누르기
2. 키보드 `1` 누르기 → n8n만 설치됩니다.
3. 나중에 `[3]` 으로 실행합니다.

> ⚠️ **Node.js를 방금 설치했다면**, 검은 창을 **완전히 닫고** `.bat` 을 **다시 실행**해 주세요. (그래야 컴퓨터가 Node.js를 새로 인식합니다.)

## 🔵 처음 실행 시 파란 경고창이 뜨면 (꼭 읽기)

인터넷에서 받은 `.bat` 파일을 **처음 두 번 누르면**, 파란색 큰 창이 뜰 수 있습니다.

> **"Windows의 PC 보호"** / "Windows protected your PC"

이건 바이러스라는 뜻이 **아니라**, "처음 보는 파일이라 한 번 확인한다"는 Windows의 안전 절차입니다. 다음처럼 진행하세요.

1. 파란 창에서 **"추가 정보"**(More info) 라는 작은 글씨를 누릅니다.
2. 아래에 **"실행"**(Run anyway) 버튼이 나타나면 그걸 누릅니다.
3. 그러면 검은 창(메뉴)이 정상적으로 열립니다. (이 과정은 보통 **처음 한 번만** 필요합니다.)

> 🛡️ **백신(보안 프로그램)이 `.bat` 을 지우거나 막는다면:** 잠깐 백신을 끄거나, 이 파일을 "예외/허용" 목록에 넣은 뒤 다시 실행하세요. 모르는 사람에게 받은 파일이 아니라면 안전합니다. (이 키트는 인터넷에서 n8n을 받아 설치할 뿐, 컴퓨터를 건드리지 않습니다.)

## ▶️ 실행 방법

1. `n8n_Manager_One-Click_Install-Run.bat` 두 번 누르기
2. 메뉴에서 `3` 누르기 (= "n8n 시작하기")
3. 잠시 뒤 브라우저가 저절로 열립니다. 안 열리면 주소창에 직접 입력: **http://localhost:5678**

**끄는 방법:** n8n이 켜진 검은 창에서 키보드 **`Ctrl` + `C`** 를 함께 누르고, "종료하시겠습니까(Y/N)?" 가 나오면 `Y` 를 누릅니다.

> 더 자세한 화면 설명·첫 로그인·워크플로우 만들기 첫걸음은 👉 [GUIDE.md](./GUIDE.md) 를 보세요.

## 🧭 메뉴 한눈에 보기

검은 창에서 **번호 키를 한 번** 누르면 됩니다(엔터 안 눌러도 됨).

| 키 | 메뉴 | 설명 |
|---|---|---|
| `0` | 처음이세요? | 설치 + 실행을 한 번에 (처음엔 이걸 추천) |
| `1` | 설치하기 | n8n을 컴퓨터에 받습니다 |
| `2` | 업데이트하기 | n8n을 최신 버전으로 새로 고칩니다 |
| `3` | 시작하기 (5678) | n8n을 켭니다 (보통 이걸 사용) |
| `4` | 시작하기 (번호 직접) | 통로(포트) 번호를 직접 정해 켭니다 |
| `5` | 백업 / 폴더 열기 | 내 작업물을 안전하게 복사해 둡니다 |
| `6` | 문제 해결 | 어디가 문제인지 자동으로 점검합니다 |
| `7` | 삭제하기 | n8n을 지웁니다 (작업물 보관/완전삭제 선택) |
| `8` | 끝내기 | 이 메뉴 창을 닫습니다 |
| `9` | 사용설명서 보기 | 글로 된 안내서를 메모장으로 엽니다 |

## ❓ 문제·오류가 생기면

1. **먼저 `[6]` 문제 해결** 을 누르세요. Node.js·n8n·인터넷 상태를 자동 점검하고, **"이럴 땐 이렇게 하세요"** 처방까지 알려줍니다.
2. **파란 경고창**(`Windows의 PC 보호`)이 막는 경우 → 위 [7번 항목](#-처음-실행-시-파란-경고창이-뜨면-꼭-읽기) 참고 ("추가 정보 → 실행").
3. 그래도 안 되면, `.bat` 을 **마우스 오른쪽 클릭 → "관리자 권한으로 실행"** 으로 다시 열어보세요.
4. 점검 결과 파일(`n8n_진단결과.txt`)을 도와줄 사람에게 그대로 보내면 빠르게 해결됩니다.

> 증상별 상세 해결법(브라우저가 안 열림, 포트 사용 중, 비밀번호 분실, 한글 깨짐 등)은 👉 [GUIDE.md](./GUIDE.md) 의 "문제·오류 대처" 장을 보세요.
